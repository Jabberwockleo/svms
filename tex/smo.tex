\title{Sequential Minimal Optimization}

\subject{Overview}
SMO is a milestone fast solver invented in [J. Platt 1998]. The most prominent contributions are
\startitemize
\item Solves sub-problems analytically, rid of the numerical QP solver library (less susceptible to numerical precision problem, compared to the original Vapnik chunking algorithm).
\item Training time scales between $O(n)$ and $O(n^2)$ in the data set size.
\stopitemize

\subject{Algorithm In Brief}
There are two components to SMO
\startitemize 
\item An analytic method for solving for the two Lagrange multipliers.
\item A heuristic for choosing which multipliers to optimize.
\stopitemize
